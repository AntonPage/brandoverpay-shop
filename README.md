## 📋 Опис проекту

Повнофункціональний інтернет-магазин преміум товарів з можливістю перегляду каталогу, додавання товарів до кошика, оформлення замовлень та адміністрування контенту.

**Концепція**: Іронічний магазин дорогих брендових товарів, де ціна = якість + переплата за бренд 

---

## 🚀 Технології

## 📦 Встановлення

### 1️⃣ Вимоги:
- XAMPP (PHP 8.0+, MySQL 8.0+)
- Git

### 2️⃣ Клонування репозиторію:
```bash
cd c:/xampp/htdocs
git clone https://github.com/YOUR_USERNAME/brandoverpay-shop.git internet-shop
cd internet-shop
```

### 3️⃣ Налаштування бази даних:
1. Запустіть XAMPP (Apache + MySQL)
2. Відкрийте phpMyAdmin: http://localhost/phpmyadmin
3. Створіть нову базу даних `internet_shop`
4. Імпортуйте файл `database.sql`

### 4️⃣ Конфігурація:
Перевірте налаштування у `config.php`:
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'internet_shop');
define('DB_USER', 'root');
define('DB_PASS', '');
define('SITE_NAME', 'BrandOverpay.com');
```

### 5️⃣ Запуск:
Відкрийте браузер: **http://localhost/internet-shop**

## 🎯 Основні Use Cases

### 1. Перегляд товарів
Користувач заходить на головну сторінку → бачить каталог товарів → може фільтрувати за категоріями

### 2. Додавання до кошика
Користувач вибирає товар → натискає "До кошика" → товар додається (AJAX)

### 3. Оформлення замовлення
Користувач переходить в кошик → натискає "Оформити" → заповнює форму доставки → підтверджує замовлення

### 4. Управління товарами (Адмін)
Адмін входить в панель → переходить в "Товари" → додає/редагує/видаляє товар

### 5. Обробка замовлень (Адмін)
Адмін бачить нові замовлення → змінює статус → клієнт отримує оновлення

## 📊 База даних (ER-діаграма)

```
┌─────────┐         ┌──────────┐         ┌──────────────┐
│  users  │────1:N──│  orders  │────1:N──│ order_items  │
└─────────┘         └──────────┘         └──────────────┘
                                                  │
                                                  │N:1
                                                  ↓
┌────────────┐                           ┌──────────┐
│ categories │──────1:N──────────────────│ products │
└────────────┘                           └──────────┘
```

### Таблиці:
- **users** - Користувачі (id, email, password, full_name, phone, role)
- **categories** - Категорії товарів (id, name, description)
- **products** - Товари (id, name, description, price, stock, category_id, image)
- **orders** - Замовлення (id, user_id, total_amount, status, delivery_address, phone)
- **order_items** - Товари в замовленні (id, order_id, product_id, quantity, price)
